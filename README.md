# TaskFlow

TaskFlow is a simple command-line task manager built in Java.

It allows users to create, manage, and track tasks with persistent local storage using file handling.

This project was built to strengthen core Java fundamentals, especially object-oriented programming and file I/O.

---

## Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using local file system

---

## Tech Stack

- Java
- Java NIO (file handling)

---

## How to Run

### 1. Compile
```bash
javac -d bin src/*.java
```

### 2. Run

```bash
java -cp out Main
```
